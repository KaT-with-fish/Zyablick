# Зяблик (Zyablick) — Многофункциональный AI Discord Бот

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![Discord.py](https://img.shields.io/badge/discord.py-API-blue.svg)
![Status](https://img.shields.io/badge/Status-Legacy-red.svg)

**Зяблик** — это мой первый крупный pet-проект, который я начал разрабатывать в 14 лет (2022).  Он задумывался как простой бот для сервера с друзьями, но со временем органично разросся, был добавлен на **100+ серверов** и бесперебойно работал почти 4 года.

> ⚠️ **Важный дисклеймер для проверяющих код:**
> Этот репозиторий — публичный архив моего самого раннего опыта программирования. Кодовая база представляет собой классический пример монолитного процедурного "спагетти-кода". 
> 
> Здесь присутствуют:
> * Управление состоянием (State management) через глобальные переменные.
> * Использование синхронных JSON-файлов вместо базы данных (что блокировало Event Loop при интенсивном I/O).
> * Отсутствие ООП и паттернов проектирования (весь бот написан без использования Cogs).
>
> **Несмотря на архитектурные ошибки, там есть чем гордится.** Именно на нем я методом проб и ошибок научился работать с асинхронностью, парсингом, потоками (threading), интеграцией API и обработкой медиа.

---

## 🚀 Ключевые возможности, которые были реализованы:

### 🧠 ИИ-собеседник с памятью
Бот умел поддерживать осмысленный диалог в контексте сервера и личных переписок.  
![Демо Чатбота](https://i.ibb.co/5gK4CJMT/image.png)
* Интеграция с **Gemini API** (а ранее с GPT-neox-20B).
* Реализована система сохранения контекста диалога (long-term memory) и промпт-инжиниринг для поддержания заданного характера.

### 🎨 ИИ-Генерация и обработка изображений
Бот умел рисовать картинки *за пол года* до выхода Midjourney. (Тут он ещё работал на VQGAN+CLIP)
![Демо генерации картинок](https://i.ibb.co/v6RVHKnz/image.png)
<!--https://i.ibb.co/B2YkPrwR/image.png-->
* Полная интеграция с API **Stable Horde** (генерация картинок по текстовому запросу).
* Использование **Pillow (PIL)** и **NumPy** для манипуляции с изображениями:
  * Создание классических "демотиваторов" на лету с автоподгоном размера шрифта.
  * Конвертация изображений в ASCII-арт.
  * Интеграция алгоритма **Seam Carving** (Content-Aware Scale) для создания эффекта "жмыха" с использованием `multiprocessing.pool.ThreadPool`, чтобы тяжелые вычисления не вешали асинхронный цикл бота.

### 🎵 Продвинутый музыкальный плеер
Асинхронный стриминг аудио с YouTube и SoundCloud.  
![Демо плеера](https://i.ibb.co/QFgd3vLX/image.png)
* Использование **yt-dlp** для получения метаданных и прямых ссылок на аудио.
* Интеграция **FFmpeg** для воспроизведения потока в голосовом канале Discord.
* Полноценная система очередей, зацикливания, перемешивания и динамический UI-плеер через `discord.ui.View` (кнопки управления прямо в чате).


### ⚙️ Модерация и утилиты
* Динамическое создание голосовых каналов ("канал, который создает каналы").
* Кастомная локализация (Русский/Английский).
* Система настроек (выбор ИИ-модели для рисования, включение TTS).
